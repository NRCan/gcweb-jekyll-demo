âŒˆ Set Border Radius

Web Pub Custom class to set border radius. Use in conjunction with WET border classes or blocks that have borders already.

Appearance
Default (4px)
.brdr-rds-sm
No radius (0px)
.brdr-rds-0
Medium (8px)
.brdr-rds-md
Large (16px)
.rounded or .brdr-rds-lg
X-Large (50px)
.rounded-xlg
Max/Round 100%

.rounded-max

Code

// No radius in a panel:
<div class="brdr-rds-0 panel panel-default">
...
</div>

// Medium (15px):
<div class="rounded-md panel panel-default">
...
</div>

// Large (25px):
<div class="rounded panel panel-default">
...
</div>

// X-Large (50px):
<div class="rounded-max panel panel-default">
...
</div>

// Max/Round (100%):
<div class="rounded-max well">
...
</div>
